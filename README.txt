Calculadora IPv4 - Redes de datos 1.
David Esteban López Castillo.

Descripción:
Esta pagina web tiene como función ser una calculadora IPv4, fue desarrollada en HTML y su lógica en JavaScript.
Se le permite al usuario poder ingresar la dirección IP en formato decimal y con su respectiva mascara, con el fin
de satisfacer las necesidades de el ejercicio propuesto por el docente, la cuales son:

- Calculo de IP binaria con sus respectivas porciones de red y host.
- Tipo de IP.
- Clase de IP.
- Calculo de máscara binaria y decimal.
- Calculo de host útiles y rango de IP's útiles.
- Calculo de IP de red binaria y decimal.
- Calculo de IP de Broadcast binaria y decimal.


Configuración:
El archivo "server.js" comprendido en la carpeta de envió, hace uso de el framework Node.js para creación de un 
servidor HTTP en donde permite la ejecución de la pagina principal "index.html" así como sus respectivos archivos 
necesarios para operación (style.css, logic.js). El servidor escucha por el puerto 80.

Requisitos:
Node.js (v21.7.3)

Como usar:
Por medio de CMD se dirige a la carpeta de envió, que comprende la pagina principal y sus archivos de operación.
Con el comando "node server.js" y podrá ver que se indicio un servidor HTTP con su respectivo link, debe dirigirse al mismo.
